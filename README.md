# Typika

[**Typika**](https://typika.ru/) is a lightweight browser extension for designers who want to inspect typography on websites faster.

It helps you identify fonts, check Cyrillic support, and open matching fonts in Google Fonts when available — without ads, clutter, or unnecessary complexity.

## Features

### Open Google Fonts matches in one click

Typika helps you quickly find whether a detected font is available on Google Fonts and open the matching page when possible.

### Cyrillic support checks

Instantly check whether a font supports Cyrillic characters — useful for multilingual interfaces, Russian-language products, and localization work.

### Fast and clean

Typika is designed to work instantly, without ads, trackers, or distracting UI.

### Made by a designer, for designers

The interface is crafted with attention to detail, visual balance, and everyday design workflow.

---

## Use cases

Typika is useful when you want to:

- identify fonts used on a website;
- inspect typography while browsing references;
- check whether a font supports Cyrillic;
- quickly open matching fonts in Google Fonts;
- collect font inspiration for UI/UX projects;
- analyze typography decisions in real products.

## Installation

1. [Download v1.0.0-beta](https://github.com/uxwisp/Typika/releases/download/v1.0.0-beta/typika-1.0.0-beta.zip)
2. Unzip the archive.
3. In Chromium-based browser, open `chrome://extensions/`
4. Click **Load unpacked** and select the unzipped folder.
