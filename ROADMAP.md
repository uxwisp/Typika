# Typika — Roadmap

## Реализовано

- Hover-тултип: family, size, weight, line-height, letter-spacing, style, transform, color, background
- Список всех шрифтов на странице при наведении на не-текстовый элемент
- Подсветка текста подчёркиванием при наведении
- Клик → копирует название шрифта в буфер
- Клик → открывает Google Fonts если шрифт найден в базе, иначе — поиск
- Строка "google fonts" в тултипе (✓ / ✗)
- Строка "cyrillic" в тултипе с проверкой поддержки кириллицы
- База данных 1908 шрифтов Google Fonts с флагом кириллицы (gf-db.js)
- Цепочка определения кириллицы: GF DB → текст на странице → opentype.js → canvas fallback
- Панель логов (720×40vh, 14px, cursor:text, кнопка "открыть полностью")
- Включение/выключение логов через контекстное меню иконки
- Состояние логов сохраняется между сессиями
- Иконка меняется при активации (серая / зелёная)
- Esc закрывает инспектор но не панель логов
